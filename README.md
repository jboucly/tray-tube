## ğŸ› ï¸ Installation

1. Clone the repository

```bash
$ git clone https://github.com/jboucly/yt-tray-app.git
```

2. Install dependencies

```bash
$ pnpm install
```

3. Build the app

```bash
$ pnpm build
```

4. Run the app

```bash
$ pnpm start
```

## Development

Run the app in dev mode with nodemon

```bash
$ pnpm dev
```

And run the vue app in dev mode

```bash
$ pnpm start:vue
```
